Ce projet a ete concue pour le traitement des images .
Voici les fonctionnalites: 
-images en niveau de gris 
-enlevement des canaux rouges ,bleux, verts -images en negatif d'images 
-transformation elementaire sur une image(Rotation,Redimensionnement,Decoupage) 
-Afficher histogramme d'une image -Manipulation du contraste et de l'intensite d'une image -Transformer de fourier d'une image -lissage d'une image
